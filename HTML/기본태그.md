# 텍스트 요소 태그
### 제목 태그
```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```
+ `<h>`태그는 제목을 표시할 때 사용
+ `<h1>`~`<h6>` 까지 단계별로 사용
### 단락 태그
```html
<p>단락</p>
``` 
+ `<p>`태그는 문단을 나타낼 때 사용

### 줄 나누기 태그
```html
<p>
    줄 나누기<br>
    태그<br>
<p>
```
+ `<br>`은 줄을 강제로 바꾸는 태그
+ `<br>`태그는 종료 태그가 없다

### 미리 정의된 서식 태그
```html
<pre>
    텍스트 서식
    서식을      그대로      출력
    서식    출력
<pre>
```
+ `<pre>`태그는 `<pre>`안의 텍스트 서식을 그대로 출력

### 수평선 태그
```html
<p>Hello World</p>
<hr>
<p>Hello World</p>
<hr>
```
+ `<hr>`태그는 수평선을 표시하는 태그
