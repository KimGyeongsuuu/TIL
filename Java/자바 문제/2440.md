# 별찍기 3
+ 첫째 줄에는 별 N개, 둘째 줄에는 별 N-1개, ..., N번째 줄에는 별 1개를 찍는 프로그램
```java
import java.util.Scanner
public class Main {
    public static void main(String[] args){
        
        Scanner in = new Scanner(System.in);
        
        int num = in.nextInt();
        
        for(int i=0; i<num; i++) { 
            
            for(int j=i+1; j<=num; j++) {
            
                System.out.print("*");
            }        
            System.out.println();            
        }
    }
}
```
### 결과
```
5  // 입력

출력
*****
****
***
**
*
```
