# 별찍기 4
``` java
import java.util.Scanner;
public class Main {
    public static void main(String[] args){
        
        Scanner in = new Scanner(System.in);
        
        int num = in.nextInt();
        
        for(int i=0; i<num; i++) { 
        	for(int j=0;j<i;j++){
        		System.out.print(" ");
        	}
            for(int j=i+1; j<=num; j++) {
            
                System.out.print("*");
            }        
            System.out.println();            
        }
    }
}
```
### 결과
```
5 // 입력

출력

***** 
 ****
  ***
   **
    *
```
